# Threads Visualization — Hotkey Reference

## Mouse Modes

Use the **Mouse toggle box** (below the legend) to switch between two mouse modes.

| Mode | LMB drag | RMB drag | Scroll |
|------|----------|----------|--------|
| **Standard** (default) | box-zoom | — | — |
| **Custom** | box-zoom | pan | zoom |

---

## Navigation (hold for smooth motion)

| Key | Action |
|-----|--------|
| `←` `→` `↑` `↓` | Pan in that direction |
| `=` | Zoom in |
| `-` | Zoom out |
| `R` | Reset view to original fit |

> All navigation keys work in both mouse modes.  
> Hold any key for continuous smooth motion.

---

## Drag Mode (Standard mode only)

| Key | Action |
|-----|--------|
| `Z` | Switch to zoom drag |
| `P` | Switch to pan drag |

---
